# @acot/acot-config

> A basic acot rule set.

## Installation

Install via npm:

```bash
$ npm install --save-dev @acot/acot-config
```

## Usage

Add `@acot` to the `extends` field of the acot config file.

```json
{
  "extends": ["@acot"]
}
```

## Supported Presets

- [@acot/acot-preset-wcag](../acot-preset-wcag/)
- [@acot/acot-preset-axe](../acot-preset-axe/)
